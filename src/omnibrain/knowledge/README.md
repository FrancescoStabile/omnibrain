# OmniBrain Knowledge Directory

Place your knowledge files here (Markdown, text, PDF).
OmniBrain will index and use them for context-aware responses.

## What to put here

- Personal notes and documents
- Project documentation
- Reference materials
- Anything you want OmniBrain to know about

## Format

Markdown files (`.md`) are preferred. OmniBrain will chunk and embed them
for semantic search via ChromaDB.
